# ウェブデザイン3級 重要用語辞典

ウェブデザイン技能検定3級の試験によく出る基本用語をまとめたWebサイトです。

## プロジェクト概要

このプロジェクトは、HTMLとCSSを使用して作成された静的なWebサイトです。用語辞典の形式で、アクセシビリティ、ネットワーク、HTML/CSSに関する重要な用語を説明しています。

## ファイル構成

```
web-design/
├── index.html    # メインのHTMLファイル
├── style.css     # スタイルシート
└── README.md     # このファイル
```

## HTML要素の使い方

このプロジェクトでは、以下のHTML要素を使用しています。

### `<li>`要素（リストアイテム）

`<li>`は、リストの各項目を表す要素です。必ず`<ul>`（順序なしリスト）または`<ol>`（順序付きリスト）の子要素として使用します。

**使用例（ナビゲーションメニュー）：**

```html
<nav>
  <ul>
    <li><a href="#accessibility">アクセシビリティ</a></li>
    <li><a href="#network">ネットワーク</a></li>
    <li><a href="#html-css">HTML/CSS</a></li>
  </ul>
</nav>
```

この例では、`<ul>`の中に3つの`<li>`要素があり、それぞれがナビゲーションメニューの項目として機能しています。

### `<dt>`要素（定義用語）

`<dt>`は、定義リスト（`<dl>`）内で用語を表す要素です。`<dd>`（定義説明）とペアで使用し、用語とその説明を構造化して表示します。

**使用例（用語辞典）：**

```html
<dl>
  <dt>ウェブアクセシビリティ</dt>
  <dd>高齢者や障害者など、心身の特性や利用環境に関わらず、誰もがウェブサイトを利用できること。</dd>
  <dt>ユーザビリティ</dt>
  <dd>特定のユーザーが、特定の目的を達成するために、どれだけ効率的かつ満足して使えるかの度合い。</dd>
</dl>
```

この例では、`<dt>`で用語を定義し、その直後の`<dd>`で説明を記述しています。複数の用語を定義する場合は、このパターンを繰り返します。

### その他の主要な要素

- **`<dl>`**: 定義リスト（Description List）のコンテナ要素
- **`<dd>`**: 定義説明（Description Definition）要素
- **`<ul>`**: 順序なしリスト（Unordered List）要素
- **`<section>`**: セクション要素（コンテンツの論理的な区切り）

## スタイリング

CSSでは、以下のようなスタイルを適用しています：

- `nav ul`に`display: flex`を使用してナビゲーションメニューを横並びに配置
- `dt`要素に太字とオレンジ色を適用して用語を強調
- `dd`要素に下線を追加して視覚的な区切りを提供

## 使い方

1. `index.html`をWebブラウザで開く
2. ナビゲーションメニューから各セクションに移動
3. 各用語の説明を確認
