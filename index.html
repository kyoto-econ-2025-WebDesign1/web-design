アクセシビリティ・UX

ウェブアクセシビリティ
高齢者や障害者など、心身の特性や利用環境に関わらず、誰もがウェブサイトを利用できること。
ユーザビリティ
特定のユーザーが、特定の目的を達成するために、どれだけ効率的かつ満足して使えるかの度合い。

ネットワーク・プロトコル

HTTP
Hyper Text Transfer Protocolの略。WebブラウザとWebサーバの間でデータを送受信するための通信規約。
DNS
Domain Name Systemの略。ドメイン名とIPアドレスを紐付けて管理する仕組み。

HTML/CSS

alt属性
画像の代わりとなるテキスト、代替テキストを入れる。画像が表示されない場合や、スクリーンリーダーによる読み上げに使用される。
外部スタイルシート
HTMLとは別ファイル（.css）にスタイルを記述する方法。サイト全体の見栄えを統一しやすい。
